# Hospital-Management-System

A Spring Boot application that performs **typical hospital operations** with **role-based access control** for different users.  
The project demonstrates secure backend development and user-friendly, interactive frontend.

---

## 🚀 Features
- Patient Management
- Appointment Scheduling
- Staff Management
- Billing and Payments
- Inventory and Pharmacy COntrol
- Role Based Access Control

---

## 🛠️ Tech Stack
- **Spring Boot**
- **Spring Cloud Netflix Eureka**
- **RabbitMQ**
- **PostgreSQL**
- **Maven**
- **Spring Data JPA**
- **React**
- **TypeScript**
- **Redux Toolkit**
- **Tailwind CSS**
- **Vite**

---

##🔐 User Roles

Hospital Admins

Receptionists/Clerks

Medical Staffs (Doctors, Nurses, Technicians)

Billing Personel

Inventory Managers
